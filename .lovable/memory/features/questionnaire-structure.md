---
name: Questionnaire Structure - 4-Game Engine
description: 32 questions (20 foundational + 12 probes), reverse scoring Q10/15/18/19, name & role capture
type: feature
---
- **Games**: Inner (Emotional), Thinking (Mental), Execution (Tactical), Leadership (Relational)
- **20 Foundational**: 5 per game, used for TEMR Radar Chart (Pillar Score = sum/25 * 100)
- **12 Probes**: 3 per game (labeled 1.1-4.3), used for 12-Trait Behavioral Audit
- **Reverse Scored**: Q10, Q15, Q18, Q19 (1↔5, 2↔4)
- **Scale**: 1 (Never) to 5 (Always)
- **User Info**: Name + Professional Role captured before questions
- **Scoring**: Pillar %, Delta Gap (max-min), Sustainability Index ((E%+T%)/2)
- **8 Archetypes**: Strategic Architect, High-Impact Connector, Resilient Operator, Adaptive Lead, Precision Specialist, Visionary Catalyst, Calm Stabilizer, Emerging Force
