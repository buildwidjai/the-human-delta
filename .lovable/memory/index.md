# Project Memory

## Core
Premium aesthetic: Navy & gold, Playfair Display headings, Inter body text.
Frame as "system calibration" (Performance Audit), never a "personality test".
4 TEMR Pillars: Tactical, Emotional, Mental, Relational.
5-point scale (1: <10% to 5: 90-100%) persistently shown on assessment pages.
Always use British English spelling (behaviour, analyse, personalised, etc.).
